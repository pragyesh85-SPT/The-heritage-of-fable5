# BANNED — The Slop Blacklist

Each item: the pattern, why it screams AI, what to do instead. "Banned" means
requires explicit justification tied to the brand/genre — not "never in the
history of design."

## Color & surface
1. **Purple/indigo-to-blue gradients** (hero backgrounds, buttons, text). The
   single strongest AI tell of 2024–2026. → Owner's brand accent on neutral
   surfaces; if no brand exists, derive one from the product's physical world
   (TIP ghee = warm golds; finance = confident deep neutrals).
2. **Dark theme by default** when nobody asked. Dark is a genre choice (dev
   tools, media), not a default. → Light-first for commerce, admin, field tools.
3. **Glassmorphism** — frosted cards, backdrop-blur, floating translucent
   panels. → Solid surfaces separated by spacing and subtle background shifts.
4. **Decorative blur blobs / mesh gradients** floating behind content. → Nothing.
   Background exists to make foreground readable.
5. **Neon-on-black dashboards** with glow effects. → Operator tools use calm
   neutrals; data gets the color, chrome stays quiet.

## Shape & effect
6. **rounded-2xl + drop shadow on everything.** Uniform maximal rounding =
   template. → One radius token, shadows only for true elevation (menus, modals).
7. **Gradient text** on headings. → Weight and size carry hierarchy.
8. **Borders everywhere** — boxes inside boxes. → Spacing groups; borders last.

## Layout
9. **Hero + three-feature-cards-with-icons + testimonial carousel + CTA band** —
   the template of a million AI landing pages. → Layout derived from the actual
   argument for THIS product: show the product itself, real numbers, real
   process, one clear action.
10. **Cards for everything.** Admin data in card grids instead of tables. →
    Genre density (SKILL gate 2): operators get tables, filters, bulk actions.
11. **Full-width centered everything** with enormous vertical padding.
    → Content-appropriate width (~70ch text), padding from the spacing scale.

## Type & icon
12. **Default Inter/Poppins with no decision made.** Fine fonts — as a CHOICE.
    → State the pairing and why it fits the brand voice.
13. **Emoji as icons** (🚀 ✨ 💡 ✅) in any professional surface. → A single
    consistent icon set (e.g. Lucide), sized to text, or no icons.
14. **Icon-per-bullet decoration** where icons add zero meaning. → Plain lists.

## Copy
15. **"Empower / Seamless / Unlock / Elevate / Supercharge / Revolutionize."**
    → Say what it does with a number or mechanism: "Create a payment link in
    two taps."
16. **"Get Started" / "Learn More" / "Submit"** buttons. → Name the action's
    object: "Generate link", "See pricing", "Place order".
17. **Fake specificity:** invented testimonials, stock-photo "team", "trusted
    by 10,000+" without basis, fake countdown urgency. → Real customers or
    nothing; trust is the product (UX-TRUST.md).
18. **Feature lists describing the UI instead of the outcome.** → Outcomes with
    evidence.

## Motion
19. **Looping/ambient animation, parallax everywhere, scroll-jacking,
    animation >300ms.** → Motion only confirms user actions, 150–250ms ease-out.
20. **Skeleton screens/spinner theater** while real states are missing. → The
    four states designed first, polish second.

## The meta-tells (how to catch NEW slop this list doesn't name yet)
- It could ship on any random startup's site unchanged (anonymity test).
- It decorates instead of informing — remove it and nothing is lost.
- It was the FIRST idea that came out with zero constraint from brand, genre,
  or content. First unconstrained output ≈ training-data median ≈ slop, by
  definition. When in doubt: constrain and regenerate.
