## VS CODE MEIN KAISE USE KAREIN

### Setup (ek baar)
1. VS Code kholo
2. Claude extension pe click karo (left sidebar mein Claude icon)
3. Koi bhi chat kholo

### `/day` use karna
```
Step 1 → Chat mein type karo:  /day
Step 2 → Enter dabao
Step 3 → Skill automatically chalegi — morning briefing aayega
Step 4 → Jo bhi karna hai, Hinglish mein likh do
         "aaj course karna hai, kshama ko inventory dena hai"
Step 5 → Sab apne aap hoga ✅
```

### `/checkcode` use karna
```
Step 1 → VS Code mein apna repo folder kholo
         (File → Open Folder → apna project folder select karo)
Step 2 → Chat mein type karo:  /checkcode
Step 3 → Enter dabao
Step 4 → Skill puchegi — multi-repo? kya check karna hai?
Step 5 → Jawab do — report ban jaayegi ✅
```

### `/feature` use karna
```
Step 1 → VS Code mein apna repo folder kholo
Step 2 → Chat mein type karo:  /feature
Step 3 → Enter dabao
Step 4 → Bolo kya banana hai:
         "discount code feature banana hai"
         "admin se price change hoga toh website pe bhi dikhe"
Step 5 → Plan dikhega — approve karo — build hoga ✅
```

### Important Tips for VS Code
- **Ek hi jagah sab kaam** — `/day` VS Code mein bhi perfectly kaam karta hai
- **Slash type karo** → `/` likhte hi saare skills ki list dikhegi
- **Kisi bhi chat mein** `/day` type karo — Notion Brain se context load ho jaayega
- **Reload karna ho** → `Ctrl+Shift+P` → "Developer: Reload Window"

## MOBILE (Claude App) MEIN KAISE USE KAREIN

### Setup (ek baar)
1. Claude app kholo (Android/iOS)
2. Left side mein **"Day Planing."** project dikhega — yahi tera PA project hai
3. Open karo

### `/day` use karna (Mobile)
```
Step 1 → Claude app mein "Day Planing." project open karo
Step 2 → New Chat pe tap karo  (+)
Step 3 → Seedha jo bhi likhna hai likh do:
         "aaj kya karna hai plan karo"
         "kal guruji ke paas ja raha hun"
         "din khatam"
Step 4 → Skill automatically samjhegi — /day likhne ki zaroorat NAHI
         (Project instructions automatically load hoti hain)
Step 5 → Sab hoga — Notion update, Calendar, Gmail ✅
```

### Mobile Pe Important Baat
- **Hamesha "Day Planing." project se kaam karo** — regular chat se mat karo
- **Naya chat = fresh /day session** — koi bhi naya chat kholo project mein
- **/day likhne ki zaroorat nahi** mobile pe — project khud samjhega
- **Kisi bhi waqt** use karo — subah, dopahar, raat — skill time samjhti hai
